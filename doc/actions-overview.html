Actions

Actions are a declarative way to change state and are triggered by events.

By using a declarative set of actions the need for scripting is massively reduced while providing better readability and maintainence.

Action Overview

Action Description
xf-action allows to group a set of actions
xf-append appends an entry to a xf-repeat
xf-delete deletes one entry from a repeat
xf-dispatch dispatches an event
xf-insert insert an item into a xf-repeat
xf-load load a document into the browser
xf-message display a message to the user
xf-refresh forces a full UI update reapplying the UI Model to the controls
xf-reset roll back all data instances to their initial state after init
xf-return returns values from an embedded form (xf-control)
xf-setindex set the index of a xf-repeat
xf-setvalue set the value of an data item
xf-submit triggers execution of a xf-submission
xf-toggle toggles a specific xf-base
xf-update force complete update of UI by passing all changes to server
