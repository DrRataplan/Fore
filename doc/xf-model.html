xf-model

The xf-model element is a required element that defines the policies of a form in a descriptive format.

It may occur anywhere in an HTML page but by convention should be put at the beginning of the body of an HTML page outside of the xf-form element.

The xf-model element and its children use HTML Custom Element syntax but have no Web Component code attached to them. These elements are never send to the browser. Instead the xf-model and its descendants are processed by eXist-db with XQuery. The Custom Element syntax is used for consistency with the UI markup when embedded in a single HTML page.
